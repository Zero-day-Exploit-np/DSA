# 🧠 1. What is Bellman-Ford Algorithm?

👉 Bellman-Ford finds the **shortest path from a single source to all vertices**.

### ✅ Special Features:

* Works with **negative weights** ✔
* Detects **negative cycles** ✔

---

# 📌 2. When to Use It?

| Algorithm    | Negative Weights | Negative Cycle Detection |
| ------------ | ---------------- | ------------------------ |
| Dijkstra     | ❌ No             | ❌ No                     |
| Bellman-Ford | ✅ Yes            | ✅ Yes                    |

---

# 🏗️ 3. Core Idea (Very Important)

👉 Repeat relaxation **(V - 1) times**

Why?

Because:

* A shortest path can have at most **V-1 edges**

---

# 🔁 4. Relaxation Concept

This is the heart of the algorithm.

```
if (distance[v] > distance[u] + weight)
    distance[v] = distance[u] + weight;
```

👉 Meaning:

* If going through `u` gives a shorter path → update it

---

# 📦 5. Your Data Structure

### You used:

```c
graph[j][0] → source
graph[j][1] → destination
graph[j][2] → weight
```

👉 This is called an **Edge List**

Better version:

```c
typedef struct {
    int src, dest, weight;
} Edge;
```

# 🚀 6. Step-by-Step Code Explanation

---

## 🔹 Step 1: Initialization

```c
for (int i = 0; i < vertices; i++)
    distance[i] = INF;

distance[source] = 0;
```

👉 Initially:

* All nodes = ∞ (unknown)
* Source = 0

---

## 🔹 Step 2: Relax Edges (V-1 times)

```c
for (int i = 0; i < vertices - 1; i++)
{
    for (int j = 0; j < edgeCount; j++)
    {
        int u = edges[j].src;
        int v = edges[j].dest;
        int w = edges[j].weight;

        if (distance[u] != INF && distance[v] > distance[u] + w)
        {
            distance[v] = distance[u] + w;
        }
    }
}
```

👉 Loop explanation:

* Outer loop → repeat V-1 times
* Inner loop → check all edges

---

## 🔹 Step 3: Detect Negative Cycle

```c
for (int j = 0; j < edgeCount; j++)
{
    if (distance[u] != INF && distance[v] > distance[u] + w)
    {
        printf("Negative cycle detected\n");
        return;
    }
}
```

👉 If still updating → cycle exists

---

## 🔹 Step 4: Print Output

```c
printf("%d \t\t %d\n", i, distance[i]);
```

# ⚡ 10. Optimization Trick

Stop early if no update:

```c
int updated = 0;

for (...) {
    updated = 0;

    if (relaxed) updated = 1;

    if (!updated) break;
}
```

👉 Saves time in many cases
